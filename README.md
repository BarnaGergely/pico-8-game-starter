## Controls

- Dpad (Arrow Keys)
    - Move
- X Button (X or M or V Key)
    - Hold X Button and Dpad: Turn
    - Release X Button: Generate or Remove Ice
- O Button (Z or N or C Key)
    - Undo (Not possible at some levels)
- Pause Menu (Enter or P Key)
    - Retry
    - Select Level
    - Delete Save Data
